Repository Overview

Smartstore — Open-Source E-Commerce
Enterprise-grade · Modular · Cross-platform · .NET 10

Smartstore is a free, open-source e-commerce platform built on ASP.NET Core (.NET 10). It delivers a complete online store — catalog, cart, checkout, payments, and order management — within a modular, plugin-based architecture. Completely rewritten for modern .NET, it supports multi-store, multi-language, and multi-currency operations. The admin features a Vue.js DataGrid, and the platform ships with 17 plugin modules covering payments, auth, analytics, and AI-powered content generation (OpenAI, Gemini).
